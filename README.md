## 🎬 Demo Features

- 🕹️  **Animated intro** — watch the guy put his glasses on and get to work
- 🖥️  **Services showcase** — ASCII computer displays every service as the guy walks and points
- 🌐 **SEO & AEO spotlight** — line-by-line reveal with 3 characters animating throughout
- 🤖 **AI Workflow simulation** — 13-step workflow with robot lifting `[WORK]` and cactus arms
- 📋 **Live clipboard** — checkboxes tick off in real time as the summary builds
- 📞 **Contact slide** — animated ringing phone
- 🏆 **Logo outro** — ASCII art shield + Chief AI Advisors sign-off

## 🚀 Getting Started

**Requirements:** Python 3.7+  — no external libraries needed.

**1. Download the demo file:**
```
chief_ai_advisors_demo.py
```

**2. Run it:**
```bash
python chief_ai_advisors_demo.py
```

**3. Follow the prompts** — press `Enter` at each screen to advance.

> 💡 Best viewed in a terminal with a monospace font (default on most systems).
